Types

Core Abstract Types

Newtrinos.Physics — Type
Physics

Abstract type for physics modules. Subtypes must have params::NamedTuple and priors::NamedTuple fields.

source
Newtrinos.Experiment — Type
Experiment

Abstract type for experiment modules. Subtypes must have physics, params, priors, assets, forward_model, and plot fields.

source

Result Type

Newtrinos.NewtrinosResult — Type
NewtrinosResult(; axes, values, meta=Dict())

Container for scan/profile results. axes holds the scan grid coordinates, values holds likelihood values and optimized parameters, meta holds execution metadata.

source

Wrapper Type

Newtrinos.Wrapper — Type
Wrapper <: Experiment

Wraps an experiment with parameter name aliases. The forward model and plot functions automatically translate between aliased and original parameter names.
